PROYECTO FINAL 3 – DASHBOARD CIENTÍFICO GENERADO 
CON PYTHON Y VARIABLES DINÁMICAS

## Descripción
Este proyecto analiza datos relacionados con el COVID-19, incluyendo defunciones, 
nuevos casos, hospitalizados y pacientes en UCI. Los datos se agrupan por día de la semana 
y por provincia, y se generan gráficas para visualizar la información.

El proyecto está dividido en tres ejercicios:
1. **Ejercicio 1**: Lectura y agrupación de datos desde un archivo CSV, cálculo de acumulados 
y almacenamiento en un archivo JSON. Agrupar datos por día de la semana y provincias y mostrar 
resultados.
2. **Ejercicio 2**: Generación de gráficas a partir de los datos almacenados, con un menú interactivo 
para visualizar las gráficas basadas en el ejercicio uno, es decir, defunciones, nuevos casos,
hospitalizados y pacientes en UCI y mostrar en gráfica los acumulados por día de la semana y 
provincias
3. **Ejercicio 3**: Crear gráficas con la provincia con el mayor número basado en los datos del 
ejercicio número uno, luego mostrar en pantalla los datos máximos y mínimos de cada provincia.


##Librerías##

- Se importa el archivo CSV y se crea e importa el archivo JSON.
- Se importa matplotlib para las gráficas.
- Se importa datetime para convertir el formato de fechas a días de la semana.

##Codificación##

- Se añade una línea "enconding='utf-8', ya que el programa no reconocía la codificación.
